About

This website is a working prototype, as the artefact for MSc dissertation project by Jiangnan Xu (s2073543@ed.ac.uk) at the University of Edinburgh.

The project aims to improve the user experience for users to browse, search and use data visualization tools.

The website is designed based on the prior version: https://vistools.net/

We recommend users to use Chrome to view the web for a best experience.
